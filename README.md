# Safevault-An-Object-Oriented-Secure-Finance-Framework

## 📌 Project Overview
SafeVault is a C++ Object-Oriented Programming (OOP) project developed as a semester final project. It is designed to provide a secure and user-friendly financial management system.

## 📂 Repository Contents
- 📄 Project Proposal: SafeVault_Proposal.pdf
- 📄 Project Report: SafeVault_Report.pdf
- 📊 Project Presentation: SafeVault_Presentation.pptx

## 🚀 Features
- User Authentication
- Secure Financial Record Management
- File Handling
- Object-Oriented Design
- Simple and User-Friendly Interface

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
